# IntelQuiz

Interactive flashcard quiz application for learning urban planning concepts.

## Features

- 40 comprehensive questions on urban planning theories
- Multiple choice and fill-in-the-blank questions
- Responsive design for mobile and desktop
- Progress tracking and results analysis
- Local storage for deck management

## Live Demo

[Deploy to Vercel](https://vercel.com/new/clone?repository-url=https://github.com/demesis221/intelquiz)

## Local Development

```bash
cd frontend
npm install
npm run dev
```

## Deployment

This app is configured for easy deployment on Vercel. Simply connect your GitHub repository to Vercel and it will automatically deploy.
